# 21EventDashboard — Информационное табло для кампусов «Школы 21» 📺

> С любовью из казанского кампуса (donyaelm) ❤️

Профессиональное решение для TV-экранов кампуса, которое автоматически транслирует события из OpenAPI Школы 21, анонсы клубов и важные объявления от АДМ. 

## ✨ Основные возможности

- **Live Sync** — автоматическое обновление данных каждые 15 минут без перезагрузки страницы.
- **Google Sheets CMS** — управление контентом (реклама, цвета, картинки клубов) напрямую через Google Таблицы. Гитхаб больше не забивается мелкими коммитами.
- **Smart Visuals** — динамические Mesh-градиенты и поддержка кастомных 3D-ассетов.
- **Multi-QR System** — автоматическая генерация QR-кодов для всех ссылок в описании с умным определением домена (`t.me`, `forms.yandex.ru`).
- **TV Optimized** — гигантская типографика, отсутствие курсора и адаптивная неоновая подсветка локаций.

## ⚙️ Управление контентом (Админка)

Все настройки экрана вынесены в Google Таблицу. Для внесения изменений просто отредактируйте нужную ячейку. Экран обновится автоматически в течение 15 минут.

### Листы таблицы:
1. **Settings** — управление таймингами и режимами отладки.
2. **TagColors** — привязка цветов к локациям (Орион, Erehwon и т.д.).
3. **ClubAssets** — привязка картинок к клубам по ключевым словам
4. **SchoolAds** — управление рекламной каруселью (текст, подзаголовок, фото).

## 🛠 Технологический стек

- **Frontend:** React + TypeScript + Styled Components
- **Animation:** Framer Motion
- **Data:** Axios + PapaParse (для Google Sheets)
- **Deployment:** Vercel (с настроенными Rewrites для обхода CORS)

## 🚀 Быстрый старт

1. Склонируйте репозиторий.
2. Создайте `.env` файл на основе `.env.example`.
3. Настройте свою Google Таблицу (обязательно "Опубликовать в интернете" как CSV).
4. Запустите проект:
```bash
npm install
npm run dev
```

## 🔐 Переменные окружения (.env)

```env
VITE_S21_LOGIN=твой_логин
VITE_S21_PASSWORD=твой_пароль
VITE_GOOGLE_SHEET_ID=ID_опубликованной_таблицы
VITE_GOOGLE_SHEET_GID_SETTINGS=...
VITE_GOOGLE_SHEET_GID_TAG_COLORS=...
VITE_GOOGLE_SHEET_GID_CLUB_ASSETS=...
VITE_GOOGLE_SHEET_GID_SCHOOL_ADS=...
```

## ⚠️ Устранение неполадок

### Ошибка 401 `invalid_grant`
Если данные в `.env` верные, но авторизация не проходит — **перезайдите в свой профиль на platform.21-school.ru** в браузере. Это обновит вашу сессию в Keycloak.
